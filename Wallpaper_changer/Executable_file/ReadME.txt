This installation file is created using pyinstaller in
raspberry pi 3 mb. So the executable is created for linux
arm. And Linux-32bit-arm bootloader is used while creating
the executable file using pyinstaller.


command used to make the executable:
$ pyinstaller --onefile main.py


OS: Raspbian GNU/Linux 9.9 (stretch) armv7l
Model: Raspberry Pi 3 Model B Rev 1.2
Kernel: 4.19.42-v7+
Shell: bash 4.4.12
CPU: ARMv7 rev 4 (v7l) (4) @ 1.2GHz
